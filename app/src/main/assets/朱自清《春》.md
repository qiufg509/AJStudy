# 🌸 春 🌸

**朱自清**

  <br/>

盼望着，盼望着，东风来了，春天的脚步近了。

一切都像刚睡醒的样子，欣欣然张开了眼。山朗润起来了，水涨起来了，太阳的脸红起来了。

![春草新绿](https://kimi-web-img.moonshot.cn/img/st3.depositphotos.com/8c86ac975afc3a3e7e9e23ee78129307f7efee42.jpg)
*小草偷偷地从土里钻出来，嫩嫩的，绿绿的*

##   <br/>

**小草偷偷地从土里钻出来，嫩嫩的，绿绿的。** 园子里，田野里，瞧去，一大片一大片满是的。坐着，躺着，打两个滚，踢几脚球，赛几趟跑，捉几回迷藏。风轻悄悄的，草软绵绵的。

![春风拂柳](https://kimi-web-img.moonshot.cn/img/5b0988e595225.cdn.sohucs.com/bc9c403c72dd366d615b09dbdc6ff7d77366aaf2.jpeg)
*"吹面不寒杨柳风"，不错的，像母亲的手抚摸着你*

##   <br/>

桃树、杏树、梨树，你不让我，我不让你，都开满了花赶趟儿。**红的像火，粉的像霞，白的像雪。** 花里带着甜味儿；闭了眼，树上仿佛已经满是桃儿、杏儿、梨儿。花下成千成百的蜜蜂嗡嗡地闹着，大小的蝴蝶飞来飞去。野花遍地是：杂样儿，有名字的，没名字的，散在草丛里，像眼睛，像星星，还眨呀眨的。

![梨花似雪](https://kimi-web-img.moonshot.cn/img/img95.699pic.com/32606c05fe94388ee2b16d4c1a32f260c23715d8)
*白的像雪，花里带着甜味儿*

  <br/>

![蜜蜂闹春](https://kimi-web-img.moonshot.cn/img/pic.616pic.com/6b9720c471e5f6faae8f77068c416d1fff9514e3.jpg)
*花下成千成百的蜜蜂嗡嗡地闹着*

##   <br/>

**"吹面不寒杨柳风"，不错的，像母亲的手抚摸着你。** 风里带来些新翻的泥土的气息，混着青草味儿，还有各种花的香，都在微微润湿的空气里酝酿。鸟儿将窠巢安在繁花嫩叶当中，高兴起来了，呼朋引伴地卖弄清脆的喉咙，唱出宛转的曲子，与轻风流水应和着。牛背上牧童的短笛，这时候也成天在嘹亮地响。

![燕子归来](https://kimi-web-img.moonshot.cn/img/pica.zhimg.com/1084506e1f737e465908957a7cd1974de91b1cd0.jpg)
*鸟儿将窠巢安在繁花嫩叶当中，呼朋引伴地卖弄清脆的喉咙*

##   <br/>

**雨是最寻常的，一下就是三两天。** 可别恼。看，像牛毛，像花针，像细丝，密密地斜织着，人家屋顶上全笼着一层薄烟。树叶子却绿得发亮，小草也青得逼你的眼。傍晚时候，上灯了，一点点黄晕的光，烘托出一片安静而和平的夜。乡下去，小路上，石桥边，有撑起伞慢慢走着的人；还有地里工作的农夫，披着蓑，戴着笠的。他们的草屋，稀稀疏疏的，在雨里静默着。

![春雨江南](https://kimi-web-img.moonshot.cn/img/picture01.52hrttpic.com/3ea69d1e58b60b7d47e6b66b443c0813aca75060.jpeg)
*像牛毛，像花针，像细丝，密密地斜织着*

  <br/>

![烟雨朦胧](https://kimi-web-img.moonshot.cn/img/k.sinaimg.cn/72055b7b15199c15c9fa8dc4723ae9eb158e48f1.jpg)
*人家屋顶上全笼着一层薄烟*

##   <br/>

天上风筝渐渐多了，地上孩子也多了。城里乡下，家家户户，老老小小，他们也赶趟儿似的，一个个都出来了。**舒活舒活筋骨，抖擞抖擞精神，各做各的一份事去。** "一年之计在于春"，刚起头儿，有的是工夫，有的是希望。

![儿童放风筝](https://kimi-web-img.moonshot.cn/img/pic.616pic.com/0b1c72d0b12d7ee37759199d6e4a6ab33abd2c2d.jpg)
*天上风筝渐渐多了，地上孩子也多了*

  <br/>

![春日嬉戏](https://kimi-web-img.moonshot.cn/img/pic.616pic.com/4fa4ac45b86da4f44bca7fb86e97210ffe672a27.jpg)
*舒活舒活筋骨，抖擞抖擞精神*

##   <br/>

春天像刚落地的娃娃，从头到脚都是新的，他生长着。

春天像小姑娘，花枝招展的，笑着，走着。

**春天像健壮的青年，有铁一般的胳膊和腰脚，他领着我们上前去。**

**💮 阅读感悟 💮**

朱自清先生的《春》是一曲对生命的赞歌。
从"盼望着，盼望着"的急切期待，
到"一切都像刚睡醒的样子"的欣喜发现；
从春草、春花、春风、春雨的细腻描摹，
到"领着我们上前去"的昂扬号召——
这不仅是江南的春景图，
更是中华民族在苦难岁月中
对光明与希望的不懈追求。

*愿我们心中永远住着一个春天。*
